# Phase 18: Spreadsheet/Table View (Excel-like)

## 🎯 Goal
Implement a tabular view for job discoveries that allows users to view data in a dense, Excel-like format for quick comparison and bulk analysis.

## 🛠️ Scope
- **UI**: 
    - Add a "Spreadsheet View" button to the sidebar navigation.
    - Create a high-performance table component with Glassmorphism styling.
    - Implement column headers: Title, Company, Score, Status, Source, Location, Date.
- **Features**:
    - Sortable columns.
    - Direct "View Report" action from the table row.
    - "Export to CSV" functionality for local data portability.
- **Integration**:
    - Ensure the table uses the same filtered data as the Grid and Kanban views.

## 📈 Success Criteria
- User can toggle to a Table view from the sidebar.
- Jobs are displayed in a responsive, sortable table.
- User can export the current filtered view to a physical CSV file.
